About Section

Welcome!

This website is about web design created by Sophie Scrima as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2025. All the text comes from the following articles located on the Dribbble website. All the images were created by me using Photoshop. The website has been created using HTML and CSS in Dreamweaver.

Articles

  • The Ultimate Guide to Choosing Fonts by Monica Galvan

  • The Ultimate Guide to Choosing Colors by Stephanie Corrigan

  • Dribbble: 11 Principles of Design by Renee Fleck

Project Description

Design and develop a visually pleasing, interactive website with multiple HTML pages, navigation connecting all the pages, featuring a cohesive color scheme, a clear visual hierarchy, a balanced layout, exciting visuals, and text. The goal is to learn how to code/develop an interactive website by applying design principles.
